# 5-fetch

Проект для изучения работы с Fetch API и JSON Server. Демонстрирует различные HTTP-запросы (GET, POST, PATCH, DELETE) для работы с API кошек.

## Описание

Этот проект содержит:

- HTML страницу с подключенным JavaScript
- JSON Server для создания REST API
- Базу данных с информацией о кошках (`db.json`)
- Класс `Api` для работы с различными HTTP-методами

## Установка

1. Убедитесь, что у вас установлен Node.js (версия 14 или выше)

2. Установите зависимости:

```bash
npm install
```

## Запуск

1. Запустите JSON Server командой `npm run db`. JSON Server будет доступен по адресу `http://localhost:3000`
2. Откройте файл `index.html` в браузере или запустите его расширением Live Server

## Функциональность

Проект демонстрирует:

- **GET запросы**: получение всех кошек и фильтрация по типу крови
- **POST запросы**: создание новых записей о кошках
- **PATCH запросы**: обновление существующих записей
- **DELETE запросы**: удаление записей

### API Endpoints

- `GET /cats` - получить всех кошек
- `GET /cats?blood_type=1` - получить кошек по типу крови
- `POST /cats` - создать новую кошку
- `PATCH /cats/:id` - обновить кошку по ID
- `DELETE /cats/:id` - удалить кошку по ID
